%  Luciferase files and analysis code from "Coordination of robust single cell rhythms in  the Arabidopsis circadian clock via spatial waves of gene expression" (BioRxiv)
doi https://doi.org/10.1101/208900s Created by Mark Greenwood (U. of Cambridge). 09/02/2018.%/images/... contains the image stacks for the individual root used in Figure 3i and Figure 3- figure supplement 1.read_images.m will read all images in a directory into a 3d stack.image_threshold.m will segment the root in each image of the 3d stack.  visualise_wave.m creates a space-time plot across longitudinal sections of the root. % The following accessory functions are required:	- amp_detrend.m detrends the amplitude of the oscillations. Ref: Edwards et al.,    	  2010. https://doi.org/10.1038/msb.2010.81	- local_regression.m detrends the baseline of the oscillations. Ref: Edwards et 	  al., 2010. https://doi.org/10.1038/msb.2010.81		- interp1gap.m performs interpolation over small threshold gaps. Chade A. Greene, 	University of Texas. Available from: https://uk.mathworks.com/matlabcentral/fileexchange/45842-interp1gap
